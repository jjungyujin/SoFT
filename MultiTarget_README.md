## Multi-Target Dataset Construction for ZS-CIR

This repository contains the dataset construction pipeline for our Zero-shot Composed Image Retrieval (ZS-CIR) framework, SoFT (Soft Filtering with Textual constraints).

### 📂 Directory Structure

```
mt_pipeline/
│
├── llm/
│   ├── multi_target_candidate/
│   ├── multi_target_selection/
│   ├── single_target_selection/
│   ├── prompt/
│   ├── generate_query.py
│   ├── extract_mt_candidates_by_*
│   ├── calculate_mt_candidates_confidence.py
│   ├── select_mt_by_threshold.py
│   └── make_single_target.py
│
└── MultiTargetDataset/
    └── datasets.py
```

### 📊 Results

- `multi_target_candidate/`
: Stores raw candidate targets extracted for each (reference image, modification text) pair.
Each query may contain multiple plausible target images reflecting inherent ambiguity.

- `multi_target_selection/`
: Contains filtered multi-target sets selected based on confidence scores.

- `single_target_selection/`
: Final single-target triplets generated by rewriting modification texts to disambiguate among multiple candidates.
